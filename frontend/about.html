About Cineskop

Cineskop is a comprehensive cinema scraper tool designed to simplify the way moviegoers access information about films showing in Niš. By aggregating data from multiple cinemas into a single database, Cineskop provides up-to-date information on movie listings and schedules through an easy-to-use web interface.

Key Features

  • Aggregates movie listings and schedules from various cinemas in Niš.

  • Features a user-friendly web app to easily access cinema schedules.

  • Updates data daily to ensure the latest movie information is available.

Technology Stack

Cineskop utilizes a combination of powerful technologies:

  • Python: For scraping cinema data and handling backend logic.

  • PostgreSQL: As the database to store all cinema data.

  • Express.js: To serve backend APIs that the frontend consumes.

  • HTML/CSS/JavaScript: For crafting a responsive and intuitive frontend.

  • NGINX: As the web server for serving the frontend application.

  • Docker: Used for containerizing and deploying the application smoothly.

Purpose and Vision

The idea behind Cineskop was born from a desire to streamline the cinema-going experience in Niš. By consolidating movie schedules into a single platform, we aim to enhance accessibility and convenience for movie enthusiasts, helping them make informed decisions about their entertainment options.
